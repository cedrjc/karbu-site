🎯 L'histoire de KarBu

Né d'un besoin réel, pas d'une idée marketing.

Pendant plus de 3 ans, j'ai suivi ma consommation sur Google Sheets, voyage après voyage, dans 3 pays différents. Ce tableau, c'était mon outil du quotidien — pratique, mais pas idéal sur mobile.

Un jour, j'ai décidé de transformer tout ce travail en une vraie application mobile. Simple, rapide, efficace. Une application que j'aurais aimé avoir dès le départ.

Il m'aura fallu de nombreuses heures de développement, de tests et d'améliorations pour en arriver là. KarBu, c'est 3 ans d'expérience terrain condensés dans votre poche.

Et le fait de l'avoir publiée est déjà ma récompense. 🙏

🔐 Respect total de votre vie privée

KarBu ne collecte aucune donnée personnelle. Pas d'email, pas de compte obligatoire. Une seule permission possible : le GPS, uniquement si vous choisissez d'utiliser le suivi de trajet, et seulement après votre accord explicite.

Vos données restent sur votre téléphone, point final.

❓ Aide & Guide d'utilisation

🚗 Démarrage rapide

  • Ouvrez KarBu et ajoutez votre premier véhicule via le bouton + Nouveau Véhicule
  • Configurez le type (thermique ou électrique), le carburant, la capacité du réservoir
  • Après chaque plein, allez sur la page Ajouter et saisissez les informations
  • Vos statistiques apparaissent automatiquement sur la page Accueil

📊 Les graphiques

  • Touchez un point sur le graphique pour voir les détails de ce plein
  • Pincez pour zoomer sur une période précise
  • Le point vert 🟢 indique la meilleure valeur, le rouge 🔴 la moins bonne
  • Appuyez sur Tableau pour voir le détail complet

🌍 Gestion des pays

  • Votre pays principal est défini à la première utilisation
  • Ajoutez jusqu'à 6 pays supplémentaires depuis la page Ajouter
  • Chaque pays a une couleur distincte dans les graphiques

💾 Données & Sauvegarde

  • Exportez vos données en CSV depuis la Configuration
  • Importez une sauvegarde existante à tout moment
  • Vos données sont stockées localement sur votre téléphone
